# Predicción Mundial 2026

Simulador de predicciones para el Mundial 2026. Aplicación React moderna y responsiva.

## Características
- Fase de grupos interactiva.
- Clasificación automática (incluyendo mejores terceros).
- Bracket de eliminación directa (Round of 32 a Final).
- Diseño moderno y responsivo.

## Tecnologías
- React
- Vite
- Lucide React
- CSS Vanilla
